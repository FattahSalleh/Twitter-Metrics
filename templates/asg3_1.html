Assignment 3 - Social Network Analysis (SNA)

Community

Modularity Score: 0.70901

Modularity measures the strength of division of a network into modules.

Networks with high modularity have dense connections between the nodes within modules, but sparse connections between nodes in different modules.

We can say that each class in the community have high interclass correlation and low intraclass correlation.

The Most Central Nodes of each Community based on Eigenvector

Eigenvector Centrality is a measure of a node's influence in a network.

A high eigenvector score means a node is connected to many nodes that have high scores themselves.

The Most Central Nodes of each Community based on Betweeness

Betweeness Centrality tests how relevant a node is for the shortest network paths.

It is a measure of how important the node is for information flow over a network.
